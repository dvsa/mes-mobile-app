Journal

This is where you can view details about your driving tests for the day. It includes the time, category, location and candidate details for each test. You can also look at test details from the previous day and plan ahead by viewing tomorrow’s tests.

If you need to swap a candidate or view more detailed information, select the candidate’s name before you start the test.

When you complete a test, your journal will update to show whether the candidate passed or failed, their route number and the questions they answered.

We will take you to your journal every time you sign in to the and when you complete a test. If you want to go to your journal at any other time, select the icon in the top left corner.

Once you’re ready to start a test, select ‘start test’.

Test times

View the test start times on the left side of the screen.

Swap a candidate

To swap a candidate, select their name and then ‘swap candidate’.

View candidate details

Select the candidate’s name to view their details. This includes their address, driving licence number, application reference number and contact information.

You must check a candidate’s details if there is an exclamation mark next to their slot.

Return to your journal

If you need to go back to the journal at any time, select ‘your journal’.

Activity codes

When you finish a test an activity code is displayed on the right side of the journal screen. This shows if the test was a pass, fail or if it was cancelled.

Show me and tell me questions

The questions that you need to ask each candidate are on the right side of the screen.

Start a test

Select the ‘start test’ button next to each candidate to start their test.

Location

The test centre location is displayed in the centre of the screen.

If you need to examine a test at a different test centre, we will display the location next to the candidate’s name.
