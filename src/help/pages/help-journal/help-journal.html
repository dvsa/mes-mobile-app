Journal

This is where you can view details about your driving tests for the day. You can also look at test details from the previous day and plan ahead by viewing tomorrow’s tests.

Each test slot includes the time, category, location and candidate details for each test. If you need to swap a candidate or view more detailed information, select the candidate’s name before you start the test.

You must check the candidate’s information if there is an exclamation mark to the left of the test slot. This is because the candidate either has a double test slot, accessibility needs or other comments that you need to be aware of.

The candidate’s information includes their name, test category, slot type, licence number, application reference and any comments. It also includes their contact details. Check the candidate’s information is correct and then select ‘back’ to return to your journal.

When you complete a test, your journal will update to show the questions they answered, their route number and activity code.

Test times

View the test start times on the left side of the screen.

Swap a candidate

To swap a candidate, select their name and then ‘swap candidate’.

View candidate details

Select the candidate’s name to view their details. This includes their address, driving licence number, application reference number and contact information.

You must check a candidate’s details if there is an exclamation mark next to their slot.

Return to your journal

If you need to go back to the journal at any time, select ‘your journal’.

Activity codes

When you finish a test an activity code is displayed on the right side of the journal screen. This will show the activity code related to the test outcome.

Show me and tell me questions

The questions that you have asked each candidate are on the right side of the screen.

Start a test

Select the ‘start test’ button next to each candidate to start their test.

Location

The test centre location is displayed in the centre of the screen.
