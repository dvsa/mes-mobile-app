Test times

View the test start times on the left side of the screen.

Swap a candidate

To swap a candidate, select their name and then ‘request a swap’.

View candidate details

Select the candidate’s name to view their details. This includes their address, driving licence number, application reference number and contact details.

You must check the candidate’s information if there is an exclamation mark to the left of the test slot. This is because the candidate either has a double test slot, accessibility needs or other comments that you need to be aware of.

Location

The test centre location is displayed in the centre of each test slot.

Activity codes

When you finish a test an activity code will show on the right side of the journal screen. The activity code reflects the test outcome.

Show me and tell me questions

When you finish a test the questions you asked the candidate will show on the right side of the journal screen.

Start a test

Select the ‘start test’ button next to each candidate to start their test.
