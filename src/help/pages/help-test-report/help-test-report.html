Test timer

You can choose to select ‘start timer’.

Test requirements

Select a mandatory requirement from the top left corner to mark it as complete. The requirement will show a blue tick when it is complete.

Vehicle checks

If you recorded a fault for the candidate’s ‘tell me’ question, the test report will show it here.

You can record any faults for the candidate’s ‘show me’ question here. If you record multiple vehicle check faults, the test report will only show the most severe fault.

Record manoeuvres

Select ‘manoeuvres’ and then the type of manoeuvre. You can select any faults for control or observations.

Record driving faults

Select and hold the fault with your finger or stylus. The test report will show the number of faults next to each type of fault.

Record serious and dangerous faults

To record a serious or dangerous fault, select ‘S’ or ‘D’ then select the fault. When you record a serious or dangerous fault an icon will show to the right of the fault.

Remove a driving fault

Select the undo icon at the top of the screen and then select the fault.

Remove serious and dangerous faults

Select ‘S’ or ‘D’, the undo icon and then the fault.
