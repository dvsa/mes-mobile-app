Start the test

When you start the test, you can record any driving faults on the test report. We will start a timer so you can check how long the test is taking.

To record a driving fault, select and hold the fault with your finger or stylus pen. We will show the number of times you record a fault next to each option. If you need to remove a fault, select ‘undo’ and then select the fault.

For serious or dangerous faults you must select ‘S’ or ‘D’ in the top left of the screen, then select and hold the fault. We will show serious and dangerous faults next to each option.

When you record a manoeuvre, select ‘manoeuvres’ and the correct option. You can record for ‘control’ and ‘observations’.

Test time

When you start the test a timer will begin so you can check how long the test is taking.

Test requirements

Make sure every candidate completes the four test requirements that are in the top left of the screen. Select and hold each requirement to mark it as complete.

Record manoeuvres

Select ‘manoeuvres’ and then the type of manoeuvres. You must select the type of manoeuvre and you can select any faults for control or observations.

Record driving faults

Select and hold the fault with your finger or stylus pen. The test report will show the number of faults next to each type of fault.

To record a serious or dangerous fault, select ‘S’ or ‘D’ then select and hold the fault. When you record a serious or dangerous fault an icon will show to the right of the fault.

Remove a driving fault

Select the undo icon at the top of the screen and then select the fault.
