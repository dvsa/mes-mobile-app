Start the test

When you start the test, you can record any driving faults on the test report. There is an option to start a timer.

To record a driving fault, select and hold the fault with your finger or stylus. For serious or dangerous faults you must select ‘S’ or ‘D’ at the top of the screen, then select the fault. The test report will show the number of driving faults and ‘S’ or ‘D’ next to each type of fault.

If you need to remove a driving fault, select ‘undo’ and then select the fault. To remove serious and dangerous faults you need to select the ‘S’ or ‘D’ button, the undo icon and then the fault.

All of the mandatory requirements are in the top left corner. You need to press a requirement button to mark it as complete. It will then show a blue tick mark.

The manoeuvres, Eco and ETA buttons are identified by blue dropdown arrows and links. When you press a button it will reveal a drop-down with available options.

Record serious and dangerous faults

To record a serious or dangerous fault, select ‘S’ or ‘D’ then select the fault. When you record a serious or dangerous fault an icon will show to the right of the fault.

Remove a driving fault

Select the undo icon at the top of the screen and then select the fault.

Remove serious and dangerous faults

Select the ‘S’ or ‘D’, then undo icon, and finally the fault.
