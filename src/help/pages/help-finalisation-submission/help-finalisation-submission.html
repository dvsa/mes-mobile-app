Route number

Use the keyboard to enter the route number.

Oral explanation comments

Use the keyboard to give any extra comments if necessary.

Weather

Select the button to describe the weather conditions.

Show me question

Press ‘select a question’ to view the list of show me questions. If you need to check the wording of any show me question select the heading.

You need to select the tick next to the question you asked the candidate.

Independent driving

Select if the candidate followed a sat nav or traffic signs.

Physical description of candidate

Use the keyboard to give a physical description of the candidate.

Add feedback

If a candidate fails their test, you must add fault feedback.

Select ‘add feedback’ and use the keyboard to enter feedback about the candidate’s faults. If you chose to complete a test later, you can select ‘edit’ to finish your feedback.

Finish the test

To end the test, select ‘submit’.

Complete the test later

If you do not have time to complete the test report before your next driving test, you can complete it later. Select ‘submit’ and then ‘defer’ on the test summary screen.
