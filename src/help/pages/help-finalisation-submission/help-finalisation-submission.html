Finish the test

After you end the debrief, you must return to the driving test centre and complete the test report. This is where you enter the route number, weather conditions, show me question and select whether the driver used a sat nav or followed traffic road signs.

You will also need to describe the driver and give an overview of their test.

Select ‘submit’ to end the test.

Complete the test later

If you do not have time to complete the test report before your next driving test, you can complete it later. Select ‘complete later’ on the test report screen.

Route number

Use the keyboard to enter the candidate’s route number.

Oral explanation comments

Use the keyboard to give an overview of the candidate’s test.

Weather

Select the button to describe the weather conditions.

Tell me question

You can see the ‘show me’ question for each candidate from your journal. You must give the result in the test summary by selecting the button.

Independent driving

Select if the driver followed a sat nav or traffic signs.

Physical description of candidate

Use the keyboard to give a physical description of the candidate.

Add fault feedback

Select ‘add feedback’ and use the keyboard to enter feedback about the candidate’s faults. You can select ‘edit’ if you need to change your feedback.

If there is a serious fault you must give feedback.

Finish the test

To end the test, select ‘submit’.
