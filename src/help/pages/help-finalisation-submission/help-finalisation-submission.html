Finish the test

After you end the debrief, you must return to the driving test centre and complete the test report. This is where you enter the route number, test overview, show me question, weather conditions and select whether the candidate used a sat nav or followed traffic road signs.

You also need to give a physical description the candidate.

Select ‘submit’ to end the test.

Complete the test later

If you do not have time to complete the test report before your next driving test, you can complete it later. Select ‘submit’ and then ‘defer’ on the test report screen.

Route number

Use the keyboard to enter the candidate’s route number.

Oral explanation comments

Use the keyboard to give any extra comments if necessary.

Weather

Select the button to describe the weather conditions.

Show me question

Press ‘select a question’ to view the list of show me questions. You need to select the tick next to the correct question and then the result from the drop-down box.

If you need to check the wording of any show me question is, select the heading.

Independent driving

Select if the candidate followed a sat nav or traffic signs.

Physical description of candidate

Use the keyboard to give a physical description of the candidate.

Add feedback to a serious or dangerous fault

Select ‘add feedback’ and use the keyboard to enter feedback about the candidate’s faults. You can select ‘edit’ if you need to change your feedback.

The app will highlight any mandatory information that you do not enter.

Finish the test

To end the test, select ‘submit’.
